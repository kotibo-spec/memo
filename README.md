# Personal Memo PWA

個人利用を目的として作成したシンプルなメモ帳PWAです。  
学習および自己使用のために作成しており、一般利用や商用利用を想定していません。

## 概要
- ブラウザ上で動作するメモ帳アプリです
- メモデータは端末内（ブラウザのローカルストレージ等）にのみ保存されます
- サーバーへのデータ送信や外部連携は行いません

## 目的
本リポジトリは、個人的な学習・検証・自己利用を目的として公開しています。  
不特定多数の利用や機能保証、継続的なサポートは想定していません。
